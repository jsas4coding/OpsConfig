# v1.0.0 (2025-12-26)

Initial release of OpsConfig - a curated and modular dotfiles collection built for operational excellence.

## Added

### Neovim Configuration
- Fast completion with blink.cmp (0.5-4ms response time)
- Fuzzy finding with fzf-lua for fast file and content search
- LSP integration with Mason-managed language servers (Bash, JSON, YAML, Lua)
- Catppuccin Frappe theme with transparent background
- ASCII-compatible icons (works on any terminal without NerdFonts)
- Intelligent auto-save with debounced saving and visual feedback
- Treesitter syntax highlighting for 40+ languages
- nvim-tree file explorer
- lualine statusline
- which-key keybinding help
- conform.nvim code formatting
- nvim-osc52 SSH clipboard integration
- noice.nvim UI improvements
- alpha dashboard
- fidget LSP progress indicator

### Bash Configuration
- Enhanced history (10,000 commands with timestamps and deduplication)
- Smart navigation (auto-cd, directory spell correction, globstar)
- SSH-aware clipboard with OSC52 passthrough for remote copy operations
- Dynamic prompt with git status, virtualenv, and server name indicators
- Single command system update (`u` updates everything)
- Server name customization
- PHP version management

### TMUX Configuration
- Vi mode with vim-style keybindings for copy mode
- OSC52 passthrough for clipboard through SSH tunnels
- True color support (full 256-color and true color)
- Optimized settings with zero escape-time for fast SSH response
- Custom prefix key (Ctrl-Space)
- Mouse support for click, scroll, and resize

### Installation System
- One-line installation script
- Automatic dependency detection and installation
- Support for Ubuntu, Debian, Fedora, CentOS, RHEL, Rocky, AlmaLinux, and Alpine
- GNU Stow-based symlink management
- Python virtual environment for pynvim
- NVM with Node.js v22 installation
- JetBrains Mono and Fira Code fonts

### Clipboard Integration
- Seamless clipboard across local and remote environments
- OSC52 escape sequence support for SSH sessions
- TMUX passthrough for nested sessions
- Support for WezTerm, Gnome Terminal, iTerm2, Alacritty, and kitty

### Update System
- `update_system` function for comprehensive updates
- Custom hooks (before_install, after_install)
- Package manager detection (apt/dnf/yum/apk)
- Composer update support for PHP projects
